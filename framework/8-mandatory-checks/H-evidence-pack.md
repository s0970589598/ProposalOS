# H｜提案證據包

🔴 提案時必附

## 為什麼必有

提案內容寫得再好，客戶老闆心裡還是會問：

- 「你做過幾個？」
- 「給我看真的東西，不要 PPT」
- 「能聯絡到你的客戶嗎？」

沒有證據 = 等於剛入行 = 客戶會壓價或不簽。

## 證據包必含

### 1. 案例（Case Studies）

每份案例至少要有：

| 項目 | 內容 |
|---|---|
| 客戶背景 | 產業、規模、痛點 |
| 解決方案 | 我們做了什麼 |
| 過程 | 時程、挑戰、轉折 |
| 結果 | 量化效益（不能只說「滿意」） |
| 客戶引述 | 客戶評語（可帶 logo / 匿名） |
| 期間 | 上線日、追蹤期 |

⚠️ 客戶 logo / 名稱要事先取得授權。

### 2. 客戶可聯絡名單（Reference）

| 用途 | 內容 |
|---|---|
| 同產業 reference | 客戶 IT、PM 願意接電話 |
| 規模相當 reference | 預算、組織規模類似 |
| 痛點類似 reference | 解過類似問題 |
| 異議處理 reference | 客戶用過後遇到問題、我們怎麼解 |

不要只給最好的 case — 要包含「曾遇到問題但解決」的 case，更有說服力。

### 3. PoC / Demo

| 形式 | 用途 |
|---|---|
| 線上 demo 環境 | 客戶可自行登入試 |
| 引導式 demo | 我方帶看、show 重點 |
| 短影片 | 5 分鐘核心功能 |
| 互動式 prototype | Figma / Storybook |
| 沙盒帳號 | 客戶試 1-2 週 |

### 4. 真實資料樣本

- 真實 dashboard 截圖（去敏感資料）
- 真實報表範例
- 真實告警通知範例
- 真實 API 回應範例

⚠️ **不准編造**截圖、log、API response — 違反 GoodLinker 全域規則。
要用真實案例去敏感資料，不要編。

### 5. 認證與證明

| 類別 | 內容 |
|---|---|
| 公司資安認證 | ISO 27001、SOC 2 Type II |
| 產業認證 | HIPAA、PCI-DSS、TISAX |
| 廠商認證 | AWS Partner、GCP Partner |
| 獎項 | 大獎、新聞、媒體報導 |
| 客戶評價 | G2、Capterra、公開推薦 |

### 6. 團隊背景

| 項目 | 內容 |
|---|---|
| 核心成員履歷 | PM、Tech Lead、資安負責人 |
| 過往專案數 | 累計、按產業 |
| 公司年資 | 成立年份、員工數 |
| 財務健全 | 看狀況揭露：年營收、客戶數、續約率 |

### 7. 文件樣本

| 文件 | 用途 |
|---|---|
| 範例提案書 | 讓客戶看完整結構 |
| 範例 SOW | 範圍、交付物 |
| 範例 SLA | 服務水準 |
| 範例 DPA | 資料處理協議 |
| 範例 RTM | 需求追溯 |
| 範例驗收書 | 上線判定 |

### 8. 反方審查的答案

針對 [playbook/red-team-questions.md](../../playbook/red-team-questions.md) 的常見問題，準備好答案 + 證明。

## 證據包包裝形式

| 場景 | 形式 |
|---|---|
| 第一次面談 | 一頁 one-pager + 3 個 case 摘要 |
| 提案中段 | 完整 case study PDF |
| 客戶法務階段 | 證書、合約樣本 |
| 客戶技術評估 | Demo 環境 + 沙盒帳號 |
| 客戶老闆裁定 | 客戶 reference call |

## 常見地雷

- ❌ 編造案例 → 一被驗就完蛋
- ❌ Reference 沒先確認 → 客戶打過去對方說「我不太記得了」
- ❌ Demo 環境塞滿 fake data → 看起來像玩具
- ❌ 用客戶 logo 沒授權 → 法律問題
- ❌ 截圖含敏感資料 → 個資外洩
- ❌ 認證過期沒更新 → 等於沒有

## 維護

證據包不是一次寫完。要持續更新：

- 每個案子上線後 3 個月，做 case study
- 每年更新 reference 名單（確認願意配合）
- 每季更新數據（累計案件、營收、續約率）
- 認證到期前 60 天啟動更新

## 與其他模組的關係

- 對應全部 12 個模組（每塊都要有證據佐證）

## 反方審查問題

- 給我 3 個同產業 reference，我直接打電話
- 你的 demo 是 fake data 嗎？真的客戶在用嗎？
- ISO 27001 證書給我看，到期日什麼時候？
- 你最大的客戶是誰？做了多久？續約嗎？
- 失敗的案例分享一下，學到什麼？
