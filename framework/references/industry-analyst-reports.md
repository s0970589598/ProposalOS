## §11. Paywall report 處理 protocol

✅ **可以 mention**：
- 「Gartner 2025 Magic Quadrant 把 Microsoft 列為 Leader」（trend mention、attribution 清楚）

❌ **不可作 primary fact source**：
- 「Gartner 預測 2028 年 80% manufacturing on hybrid cloud」（具體數字、需 primary access 確認）

**處理方式**：
1. Cite secondary blog / press release 時必標 `❌ Paywall secondary`
2. 對外 deliverable 寫具體數字一律加 disclaimer「per Gartner 2025 secondary cite via Microsoft blog 等 vendor 二手引用、原報告 NDA / paywall」（實際引用時填寫 secondary source 的 URL）
3. 若客戶要 primary cite、引導客戶或自己購買 Gartner subscription、不要 fake primary access
