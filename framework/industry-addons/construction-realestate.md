# 產業加掛｜營建 / 不動產 / 工程

建設公司、營造、不動產開發、工程顧問、物業管理。

## 對應 13 模組的擴充

### 模組 01 產業

- 房市循環（首購、自住、投資、商辦）
- BIM（Building Information Modeling）普及
- 智慧建築 / 智慧城市
- 綠建築（LEED、EEWH、WELL）
- 預鑄、模組化建築
- 缺工、缺料、地緣政治影響
- 不動產代幣化（RWA）
- ESG 與低碳建材

### 模組 03 問題

| 場景 | 痛點 |
|---|---|
| 工地進度追蹤 | 紙本日報、進度落後不知 |

| 預售 / 銷售 | 接待中心、線上看屋 |
| 法規申辦 | 建管、消防、地政流程繁瑣 |

## 營建特有議題

### BIM 整合

- **LOD**（Level of Development）100-500
- IFC 跨平台格式
- 衝突偵測（Clash Detection）
- 4D（時間）/ 5D（成本）/ 6D（永續）BIM
- 跨團隊（建築師 / 結構 / 機電 / 承包）協作

### 工地實名制

- 法定（部分國家 / 地區）
- 出勤、安全教育、職災追溯
- 對接政府平台

### 安全衛生

- ISO 45001
- 職安署規範
- 重大事故 24h 通報
- 安全教育時數

### 物業管理整合

- 智慧鎖、門禁
- 訪客管理
- 報修系統
- 公共設施預約
- 住戶 App

### 預售屋戰略

- 接待中心 VR / AR
- 線上選戶 / 看屋
- 銷售合約電子化
- 客戶旅程追蹤

### 公共工程 vs 私人工程

| | 公共工程 | 私人工程 |
|---|---|---|
| 採購 | 政府採購法 | 業主自由議定 |
| 文件 | 嚴格、SOP | 較彈性 |
| 驗收 | 多階段勘驗 | 業主驗 |
| 工期罰款 | 嚴格 | 視合約 |

→ 公共工程提案必看 [Shipley](../methodologies/shipley-apmp.md) + [Compliance Matrix](../../templates/compliance-matrix-template.md)。

## 營建提案 anti-patterns

| AP | 內容 |
|---|---|
| 沒理解 BIM LOD 等級就承諾交付 | LOD 200 vs 400 工作量差 5 倍 |
| 工地網路 / 設備假設過樂觀 | 工地網路通常很爛、要設計離線 |
| 沒處理工人手機 / 電腦使用習慣 | 工人不用 PC、要設計給手機優先 |
| 沒對接政府平台 | 工地實名制、實價登錄等 |
| 多包商整合假設順利 | 包商 IT 能力差異大、難整合 |
