# 產業加掛｜製造業

GoodLinker 主場。多數案件都會用到。

## 對應 13 模組的擴充

### 模組 01 產業與市場

- 智慧製造 / 工業 4.0 / Industry 5.0 趨勢
- 美國、台灣、東南亞、中國的供應鏈轉移
- 缺工、缺料、淨零、地緣政治
- 客戶在價值鏈的位置（OEM / ODM / 品牌）

### 模組 03 問題定義

製造業常見痛點（人機料法環視角）：

| 維度 | 典型痛點 |
|---|---|
| 人 | 老師傅退休、新人訓練長、巡檢人力高 |
| 機 | 稼動率低、預知保養不準、故障停線 |
| 料 | 原料追溯、批次管理、庫存週轉 |
| 法 | SOP 不落地、品管欠數據、變更難追 |
| 環 | 能耗高、ESG 壓力、廢棄物 |

### 模組 04 流程與服務藍圖

製造業流程典型分層：

- 訂單 → 生管 → 製造 → 品管 → 出貨
- 設備層 → 線體層 → 廠級 → 集團
- 班次（早 / 中 / 晚 / 假日）

### 模組 06 資料整合

製造業特有資料源：

| 資料源 | 通訊協定 |

| SCADA | OPC UA |
| MES | API、DB、CSV |

| 電錶 / 流量計 | Modbus、Pulse |
| 感測器 | LoRa、Zigbee、WiFi、4G |
| Vision | RTSP、HTTP |

### 模組 08 系統架構

製造業常見架構分層（Purdue / ISA-95 Model）：

```
Level 4  企業 / 業務規劃（ERP / BI、時間尺度：週 / 月）
Level 3  製造運營管理（MES、時間尺度：時 / 日）
Level 2  監控（SCADA / HMI、時間尺度：秒 / 分）
Level 1  基本控制（PLC / DCS、時間尺度：毫秒 / 秒）
Level 0  物理製程（感測器 / 致動器 / 設備）
```

現代版本常加 Level 3.5 / DMZ 做 IT/OT 隔離。

跨層整合、跨層資料流向、跨層權限要寫清楚（OT 越往下越關鍵、回不去）。

### 模組 11 驗收與效益

製造業常用 KPI：

- OEE（Overall Equipment Effectiveness）= 稼動率 × 速度 × 良率
- MTBF / MTTR（平均故障間隔 / 平均維修時間）
- 良率 / 不良率
- 換線時間 / SMED
- 能耗 kWh / 產出單位
- 巡檢人時 / 月

## 製造業特有議題

### IT / OT 分離

- IT 網段、OT 網段、DMZ
- 不能讓 IT 攻擊面波及 OT
- 廠級防火牆、單向 Diode

### 工業環境

- 高溫、油污、粉塵、震動
- 工業等級設備（IP65+）
- 防爆區域（化工、油氣）

### 班次與 24/7

- 上線時程要避開高峰
- 訓練要覆蓋三班
- SLA 要 24/7

### 設備供應商

- 機台廠商（東元、台達、Siemens、Mitsubishi）
- PLC 廠商
- SCADA 廠商
- ERP / MES 廠商（鼎新、SAP、Oracle）

## 反方審查問題

- 你的系統碰 OT 嗎？資安怎麼隔離？
- 機台原本廠商不開資料怎麼辦？
- 我們有三班，訓練怎麼排？
- 停產一天損失 N 萬，你的上線怎麼避免停線？
- ESG 報告我們明年要交，你的系統可以產嗎？
