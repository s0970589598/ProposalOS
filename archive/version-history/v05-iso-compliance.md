# V5｜ISO / 法規 / 競品版

## 主題

V1-V4 缺合規與競品分析，補上。

## 核心新增

### ISO 與法規

| 類別 | 代表 |
|---|---|
| 通用資安 | ISO 27001、SOC 2、NIST CSF |
| 個資 | 個資法、GDPR |
| 品質 | ISO 9001、IATF 16949 |
| 環境 | ISO 14001、ESG |
| 醫療 | HIPAA |
| 金融 | PCI-DSS |
| 食品 | HACCP、ISO 22000 |
| AI | NIST AI RMF、EU AI Act、ISO 42001 |

### 競品分析（V5 版）

當作「提案末尾的銷售比較工具」。

V10 把這個視角修正成「需求探索工具」。

## 對應到 V12

→ ISO 與法規進入 [模組 07](../../framework/12-modules/07-compliance-security-ai-risk.md)
→ 競品進入 [模組 02](../../framework/12-modules/02-competitors-alternatives.md)，但**位置前移**

## 為什麼被 V10 修正

V5 的競品位置是「提案末尾」：

```
... 我們的方案
... 競品比較（V5 在這裡）
... 報價
```

V10 修正成「需求探索前期」：

```
產業背景 → 競品（V10 在這裡）→ 訪談 → ...
```

## 保留的價值

- 完整 ISO / 法規清單仍在模組 07
- 競品內容保留，只是位置改變
- 第一次把「合規」當作獨立大項提出
