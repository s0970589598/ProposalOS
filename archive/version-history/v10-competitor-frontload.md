# V10｜競品前移版

## 主題

V5 的競品位置錯了，要前移。

## 核心修正

| V5 / V1-V8（後置）| V10（前移）|
|---|---|
| 提案末尾「我們 vs A 公司 vs B 公司」| 訪談前就盤點客戶選項 |
| 銷售視角 | 客戶視角 |
| 把對方講爛來凸顯自己 | 借競品幫客戶找出真需求 |
| 容易被客戶看穿 | 客戶會覺得你懂這個市場 |

## 新的流程順序

```
產業背景
   ↓
競品 / 類似產品 / 替代方案  ← V10 前移到這
   ↓
初步市場功能地圖
   ↓
需求訪談
   ↓
... 後續流程
```

## V10 引入的「三類選項」

1. **直接競品**：做同樣事情的廠商
2. **類似產品**：做類似事情但定位不同
3. **替代方案**：自做 / 維持現狀 / 外包

V12 模組 02 完整保留這個分類。

## 為什麼這個修正很關鍵

很多顧問把競品當作「最後比一比」。
V10 的洞察：**競品分析是探索客戶需求的最強工具**。

客戶看過 A、B、C 都沒選 → 你可以問為什麼 → 反推真需求。
這個動作如果留到提案末尾才做，已經失去訪談機會。

## 保留的價值

- 流程順序修正
- 三類選項分類
- 「需求探索 vs 銷售比較」的視角區分
